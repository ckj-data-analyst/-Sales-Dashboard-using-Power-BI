# 📊 Sales Dashboard using Power BI

## 📌 Project Overview

This project analyzes sales performance using Power BI to generate insights on revenue, profit and customer trends.

## 🎯 Objectives

* Analyze total sales and profit
* Identify top regions and categories
* Track trends over time

## 📁 Dataset

* Superstore Dataset
* Includes Sales, Profit, Region, Category

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX
* Excel

## 🧹 Data Cleaning

* Removed null values
* Trimmed spaces
* Changed data types

## 📊 Dashboard Features

* KPI Cards (Sales, Profit)
* Bar Chart (Region-wise Sales)
* Donut Chart (Category-wise Sales)
* Line Chart (Profit Trend)
* Slicers (Region, Date)

## 📈 Key Insights

* West region generated highest sales
* Technology category contributed maximum revenue
* Profit shows seasonal variation
